# HTML-and-CSS-Course
This repository contains a high-performance, uniquely styled portfolio website for **Hasan-AlJarida**. The design follows a tech-focused, cyberpunk aesthetic featuring neon accents and dark mode optimization.

## Project Scope
This project demonstrates proficiency in HTML5 and CSS3, specifically focusing on layout management, media embedding, and interactive styling.

## Technical Implementations
* **Video Integration**: Implements a muted, looping background video with a dynamic text overlay.
* **Fixed Navigation**: A top-bar menu with custom hover transitions and anchored positioning.
* **Two-Column Architecture**: A responsive flexbox-based layout for skill and experience presentation.
* **CSS Image Processing**: 
    * Images utilize `filter` properties for a stylized tech look.
    * Custom `border-radius` for unique edge shaping.
    * Image overlay fade effects for interactive project galleries.
* **Data Representation**: HTML tables styled with specific padding, borders, and header background colors.
* **Interactive Typography**: Paragraphs (`<p>`) feature specific hover states to improve user engagement.

## Bootstrap Project
In addition to the custom HTML/CSS portfolio, this repository tracks the progress of my **Bootstrap Integration Project**, focusing on mobile-first development and grid systems.
